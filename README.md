# 可视化demo 
* 一个简单的可视化demo，用到的了 vue3 + ts + axios + echarts + pinia 🍍
* 接口数据请求后端接口获取，后端用的express 处理 ✅



## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
